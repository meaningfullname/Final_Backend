Welcome to the Combined Project Dashboard

This platform brings together a suite of web applications developed during our Web Technologies 2 course. Here, you can explore multiple features integrated into one unified system.

Project Features

  • Send Email: Easily send customized emails directly from the platform.
  • BMI Calculator: Calculate your Body Mass Index quickly and efficiently.
  • QR Code Generator: Generate QR codes for any URL or text input.
  • Blog System: Create, manage, and delete blog posts with ease.
  • Weather App: Get real-time weather updates, forecasts, and alerts for your city.

Use the navigation menu above to explore each feature. This project is built with Express.js and integrates multiple APIs and technologies to deliver a full-stack web application experience.

Thank you for visiting! We hope you enjoy exploring all the functionalities available on this dashboard.
